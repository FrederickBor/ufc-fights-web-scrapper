# UFC Data Web Scrapper

This web scrapper is used to get all the figths on Sherdog's website

## Requirements:

<center>

| Requirement Name | Version |
| ---------------- | ------- |
|beautifulsoup4|4.8.1|
|certifi|2019.11.28|
|chardet|3.0.4|
|idna|2.8|
|requests|2.22.0|
|soupsieve|1.9.5|
|urllib3|1.25.7|

```bash
pip install -r requirements.txt
```

